Desarrollo de Software

Análisis

Determinar los requisitos de un producto o software debe ser la primer etapa para su desarrollo, esta etapa es tal vez la más importante ya que si no se hace un buen “levantamiento de información” al realizar los diferentes tipos sistemas de recolección de la información, se puede desarrollar un software totalmente diferente al que solicito el cliente, para el desarrollo de esta etapa se puede afianzar sobre diferentes métodos de recolección de la información como lo son las encuestas, entrevistas, observaciones, simulaciones, técnicas audiovisuales y de inspección.

Drake, J. M. (2008). Análisis de requerimientos y especificación de una aplicación. Recuperado el 04 de 2017, de Computadores y tiempo real: url=http://www.ctr.unican.es/asignaturas/Ingenieria_Software_4_F/Doc/M3_08_Especificacion-2011.pdf
