Desarrollo de Software

Diseño y Desarrollo

Diseño:
Proceso mediante el cual se organiza la toda la información recolectada en el periodo de análisis y se estructura para crear el cronograma del desarrollo, se crea un modelo o las especificaciones para el producto a crear o las especificaciones del sistema.


Desarrollo: Consiste en crear o desarrollar el software utilizando los modelos creados durante la etapa de diseño, esto permitirá crear los componentes del sistema, esta etapa no es la más larga, todo depende de los lenguajes de programación que se vayan a implementar en el producto y que tan bien fue definido todos los modelos o casos a usar.

Drake, J. M. (2008). Proceso de desarrollo de aplicaciones de software. Recuperado el 04 de 2017, de Programación orientada a objetos: Lenguaje, Metodologias, y Herramientas: http://www.ctr.unican.es/asignaturas/MC_OO/Doc/OO_08_I2_Proceso.pdf
